# 健康管理微信小程序项目文档

## 项目概述
基于微信小程序的健康管理系统，主要功能包括：
- 用药提醒管理（添加/编辑/删除）
- 健康数据记录与展示
- 医疗报告生成
- 用户中心管理

## 技术栈
- 微信小程序原生开发
- WXML/WXSS/JS
- 使用wx-charts进行数据可视化
- RESTful API交互

## 目录结构
```
├── pages                  # 页面目录
│   ├── medication         # 用药提醒模块
│   ├── healthReport       # 健康报告模块
│   └──...
├── services               # 服务层
│   └── api.js             # 接口封装
├── utils                  # 工具类
│   ├── util.js            # 通用工具
│   └── wxcharts-min.js    # 图表库
└──...
```

## 运行方式
1. 微信开发者工具导入项目
2. 配置appid
3. 编译运行

## 注意事项
1. 需配置合法域名
2. 微信基础库版本需≥2.0
